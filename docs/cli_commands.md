# CLI Commands
## `show`
Shows the results of a benchmark from a json file.
> [!NOTE]
> This file must be generated by `BenchmarkRunner`.

Example:
```sh
py -3 -m perf_runner show bm.json
```
You can add more than one json file, it is space separated.

## `compare_to`
Compares the results of two benchmarks.
> [!NOTE]
> Both files must be generated by `BenchmarkRunner`.

Example:
```sh
py -3 -m perf_runner compare_to bm1.json bm2.json
```
